# EepDoop-Bypasser

detects visual effects (vfx) and place them in game.ReplicatedStorage.EepDoop

## Usage

- Starfall Endowment (10M)

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/xu0blu/EepDoop-Bypasser/refs/heads/main/starfall-endowment.luau"))()
```

- Every Effect (5K-9M)
  
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/xu0blu/EepDoop-Bypasser/refs/heads/main/every-effect.luau"))()
```
